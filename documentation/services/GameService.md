## getGameInformation

Retrieve game information.

- HTTP Method: `GET`
- Endpoint: `/{lang}/game`

**Parameters**

| Name | Type   | Required | Description   |
| :--- | :----- | :------- | :------------ |
| lang | string | ✅       | Language code |

**Return Type**

**Example Usage Code Snippet**

```typescript

(async () => {
```
